Linux进阶学习

发表于 2017-01-17 |

VI编辑器

Visual Interface可视化接口
VI->VIM
-VIM支持多级撤销
-VIM可以跨平台运行
-VIM支持语法高亮
-VIM支持图形界面

VI编辑器的操作模式

Commad Mode-命令模式
Insert Mode-输入模式
Last Line Mode-底行模式
vim + abc打开文件后光标在最后一行
vim +3 abc打开文件光标在第三行
vim +/xxx abc打开文件光标在xxx单词第一次出现的那一行，按n可以切换到下一个符合条件的行
vim aa bb cc同时打开三个文件 n切换到下一个文件N切换到上一个文件

命令模式常用指令

h 光标左移
j 光标下移
k 光标上移
l 光标右移
ctrl+f向上翻页（front）
ctrl+b向下翻页（back）
ctrl+d向下翻半页（down）
ctrl+u向上翻半页（up）
dd 删除光标所在行
o 在光标所在行的下方插入一行并切换到输入模式
yy 复制光标所在行
p 在光标所在行下方粘贴
P 在光标所在行上方粘贴

磁盘管理基本命令

df 查看磁盘分区的使用状况
-l 仅显示本地磁盘（默认）
-a 显示所有文件系统的磁盘使用情况，包含比如/proc/
-h 以1024进制计算最适合的单位显示磁盘容量
-H 以1000进制计算最适合的单位显示磁盘容量
-T 显示磁盘分区类型
-t 显示指定类型文件系统的磁盘分区 如ext4
-x 不显示指定类型文件系统的磁盘分区

du 统计磁盘上文件的大小
-b 以byte为单位统计文件
-k 以KB为单位统计文件
-m 以MB为单位统计文件
—h 以1024进制统计文件
-H 以1000进制统计文件
-s 指定统计目标

MBR分区

必须对硬盘分区、格式化、挂载后才能使用。扩展分区需要添加逻辑分区,
最多只能有1个扩展分区和主分区，1-4是预留给扩展分区和主分区的。
fdisk -l查看硬盘分区情况
fdisk /dev/sdb 进入分区模式
n 添加分区
p 列出分区情况
d 删除分区

p 主分区
e 扩展分区
Partition number分区号
First cylinder开始位置
Last cylinder 结束位置
分区完成后 w 写入磁盘

GPT分区

硬盘分区与格式化

分区模式之MBR

- 主分区不超过4个
- 单个分区容量容量最大2TB

分区模式之GPT

- 主分区的个数“几乎”没有限制
- 耽搁分区的容量“几乎”没有限制

使用parted命令进行GPT分区或MBR分区
select切换目标磁盘
select /dev/sdc
首先要指定分区表
mklabel msdos（MBR分区）
mklabel gpt （GPT分区）
print查看当前硬盘的分区详情
mkpart 添加分区
start? 1
end? 2000
mkpart test 2000 3000
rm 删除分区
quit 完成后退出

分区格式化

mkfs.ext3 /dev/sdb1

挂载

默认的挂载目录/mnt
mount /dev/sdc1 /mnt/test
unmount /mnt/imooc 卸载挂载
以上重启后会失效要自动挂载需编辑/etc/fstab
/dev/sdc1 /mnt/test ext3 defaults 0 0

为硬盘添加swap交换分区

第一，建立一个普通的Linux分区
第二，修改分区类型的16进制编码
第三，格式化交换分区
第四，启用交换分区
fdisk ／dev/sdb
t 改变分区属性
L 查看16进制编码

格式化交换分区

mkswap /dev/sdb6

启用交换分区

swapon /dev/sdb6

用户和用户组

用户：使用操作系统的人
用户组：具有相同系统权限的一组用户
/etc/group 存储当前系统中所有用户组信息
Group： x ：123:abc, def,xyz
组名称：组密码占位符：组编号：组中用户名列表
／etc/gshadow 存储当前系统中用户组的密码信息
Group: : :abc, def,xyz
组名称：组密码：组管理者：组中用户名列表
密码为空或者或者！代表无密码
/etc/passwd 存储当前系统中所有用户的信息
user: x : 123 : 456 :xxxxxxx:home/user:/bin/bash
用户名：密码占位符：用户编号：用户组编号：用户注释：用户主目录：shell类型
/etc/shadow 存储用户的密码信息

操作用户组

添加用户组

groupadd [用户组名称]
groupadd zhangyuang
groupadd -g 888 boss

修改名称

groupmod -n market zhangyuang

修改组编号

groupmod -g 668 market

删除用户组

groupdel market

操作用户

添加用户到用户组

useradd -g market zhangyuang
useradd -d /home/xxx zhangyuang 将用户放入指定文件夹不写的话自动创建同用户名文件夹

给用户添加备注

usermod -c xxx zhangyuang

更改用户名

usermod -l newname zhangyuang

修改用户所属用户组

usermod -g newgroup zhangyuang

删除用户

userdel zhangyuang
userdel -r zhangyuang //同时删除个人文件夹

禁止除root外普通用户登录

touch /etc/nologin

锁定用户

passwd -l zhangyuang

解锁账户

passwd -u zhangyuang

清除用户密码

passwd -d zhangyuang

添加附属组

gpasswd -a zhangyuang othergroup

切换身份到附属组

newgrp othergroup

删除附属组

gpasswd -d zhangyuang othergroup

创建用户带有附属组

useradd -g maingroup -G othergroup

修改组密码

gpasswd othergroup

切换用户身份

su username

显示指定用户信息，包括编号，用户名

id zhangyuang

显示用户所在组

groups zhangyuang

设置用户资料

chfn zhangyuang

显示用户详细资料

finger zhangyuang
