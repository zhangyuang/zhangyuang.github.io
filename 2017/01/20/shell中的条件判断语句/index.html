shell中的条件判断语句

发表于 2017-01-20 |

按照文件类型进行判断

-b 文件：判断该文件是否存在，并且是否为块设备文件（是块设备文件为真）
-c 文件：判断该文件是否存在，并且是否为字符设备文件（是为真）
-d 文件：判断该文件是否存在，并且是否为目录文件（是目录为真）

-e 文件：判断该文件是否存在（存在为真）
-f 文件：判断该文件是否存在，并且是否为普通文件（是普通文件为真）
-L 文件：判断该文件是否存在，并且是否为符号链接文件（是为真）
-p 文件：判断该文件是否存在，并且是否为管道文件（是为真）
-s 文件：判断该文件是否存在，并且是否为非空（非空为真）
-S 文件：判断该文件是否存在，并且是否为套接字文件（是为真）

两种判断格式

test -e /root/install.log
[-e /root/install.log] && echo “yes”||echo “no”

#第一个判断命令正确执行打印yes否则打印no

按照文件权限进行判断

-r 文件：是否存在，并且是否该文件拥有读权限（有读权限为真）
-w 文件：是否存在，是否拥有写权限
-x 文件：是否存在，是否拥有执行权限
-u 文件：是否存在，是否拥有SUID权限
-g 文件：是否存在，是否拥有SGID权限
-k 文件：是否存在，是否拥有SBit权限

两个文件之间进行比较

文件1 -nt 文件2 判断文件1的修改时间是否比文件2的新（如果新则为真）
文件1 -ot 文件2 判断文件1的修改时间是否比文件2的旧
文件1 -ef 文件2 判断文件1和文件2的inode号是否一致，可以理解为两个文件是否为同一个文件。这个判断用于判断硬链接是很好的方法
ln /root/student.txt /tmp/stu.txt

#创建硬链接
[ /root/studeng.txt -ef /tmp/stu.txt ] && echo “yes” || echo “no”

#用test测试

两个整数的比较

整数1 -eq 整数2 判断整数1和整数2是否相等
-ne 不相等
-gt 整数1是否大于整数2
-lt 是否小于
-ge 是否大于等于
-le 是否小于等于

字符串的判断

-z 字符串 判断字符串是否为空（为空返回真）
-n 字符串 判断是否为非空
字符串1==字符串2 判断是否相等
字符串1！=字符串2 判断是否不相等

多重条件判断

判断1 -a 判断2 逻辑与，判断1和判断2都成立，为真
判断1 -o 判断2 逻辑或，判断1和判断2有一个成立最终的结果就为真
！判断 逻辑非，使原始的判断式取反

if语句

if [条件判断式]； then
程序
fi
或者
if [条件判断式]
then
程序
fi

例子1 判读登录的用户是否root

#!/bin/bash
c
if [ "$test"==root]
    then
    echo "user is root"
fi

例子2 判断分区使用率

#!/bin/bash
rate=$(df -h | grep "/dev/sda3" | awk'{print $5}' |cut -d "%" -f 1)
if [ $rate -ge 80]
    then 
    echo "warning /dev/sda3 is full"
fi

双分支if语句

if [条件判断式]
then
条件成立时，执行的程序
else
条件不成立时，执行的程序
fi

判断是不是一个目录

#!/bin/bash
read -t 30 -p "please input a dir" dir
if [ -d "$dir" ]
    then
    echo "is a dir"
else 
    echo "is not a dir"
fi

判断apache服务是否开启

ps aux查看所有进程

#!／bin/bash
test=$(ps aux | grep httpd | grep -v grep)
if [ -n "$test"]
    then
    echo "$(date)httpd is ok">>/tmp/autostartacc-log
else
    /etc/rc.d/init.d/httpd start &> /dev/null
    echo "$(date)restart httpd">>/tmp/autostart-err.log

多分支if

if [条件判断式1]
then
成立时，执行程序1
elif [条件判断式2]
then
成立时，执行程序2
else
所有条件都不成立时执行此程序
fi

case语句

case $变量名 in
“值1”)
程序1
;;
“值2”)
程序2
;;
*)
如果都不是，执行此程序
;;
esac 

#!/bin/bash
read -p "please choose yes/no:" -t 30 cho
case $cho in
    "yes")
        echo "is yes"
        ;;
    "no")
        echo "is no"
        ;;
    *)
        echo "is error"
        ;;
    esac

for循环

语法1
for 变量 in 值1 值2
do
程序
done

#!/bin/bash
for i in 1 2 3 4 5    
    do
        echo $i
    done

批量解压缩

#!/bin/bash
cd /root/test
ls *.tar.gz > ls.log
ls *.tgz >> ls.log
for i in $( cat ls.log )
    do
        tar -zxvf $i &> /dev/null
    done
rm -rf ls.log

语法2
for((初始值；循环控制条件；变量变化))
do
程序
done
