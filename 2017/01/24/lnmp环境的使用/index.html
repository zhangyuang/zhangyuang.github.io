lnmp环境的使用

发表于 2017-01-24 |

之前的项目是放在阿里云centos服务器上的当时不太懂Linux就叫别人帮我装了一个Lamp环境都是别人在搞。最近学了Linux之后就想把之前的项目在自己的虚拟机服务器上跑一遍。由于Lnmp的性能和安装便捷性都高于Lamp所以就学习lnmp了。lamp:Linux apache mysql php lnmp：Linux nginx mysql php

在www.lnmp.org里有一键安装的脚本所以安装过程我就不说了，我这里说说安装后如何把网页放上去以及安装tomcat来接通后台的借口。
网页文件默认放在/home/wwwroot/ default下我的项目文件全部放在Hg_project文件夹下所以访问方式


安装tomcat在http://tomcat.apache.org/download-80.cgi下载，我下载的是tomcat8。下载完后检查服务器是否安装java jdk，java -version显示版本的话说明安装了

没有安装的话http://www.cnblogs.com/hanyinglong/p/5025635.html参考该地址进行安装。
解压下载的tomcat压缩包mv命令改名为tomcat，然后mv命令移动目录到/usr/local/tomcat下。成功后地址栏输入ip:8080.出现


安装成功。
接着借口文件放在/usr/local/tomcat/webapps的目录下

最后成功访问

是500的原因是因为我只改了一个js文件的接口地址。反正不是404就行= =
